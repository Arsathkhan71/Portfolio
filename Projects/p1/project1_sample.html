FaceID: Advanced Facial Recognition and Object Detection System

FaceID is a cutting-edge system for facial recognition and object detection. It uses a color-coding system to identify individuals and YOLO for real-time object detection. Data is efficiently managed with PostgreSQL and visualized through a user-friendly dashboard.

Key Features

  • Facial Recognition: Detects and recognizes faces with precision using a color-coded system.
  • Object Detection: Employs YOLO for real-time identification and tracking of various objects
  • Data Management: Uses PostgreSQL for structured data storage.
  • Interactive Dashboard: Provides clear insights with HTML, CSS, and Flask, enhanced by PowerBI for analytics.
